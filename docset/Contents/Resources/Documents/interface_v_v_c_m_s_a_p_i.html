VVCMSAPI Class Reference

Description

VVCMSAPI is used to query video records managed by a VolarVideo CMS.

When instantiating, you must provide a valid domain and API key. Your API key will affect the scope of content you have access to. The same request could yield different results for different API keys.

Every request should be provided a VVCMSAPIDelegate to handle responses. Work is done on another thread and delegate methods are not guaranteed to be called on the same thread they were requested from.

It's important to call shutdown when you are finished with an instance of VVCMSAPI. This will prevent any unfinished requests from calling their delegate methods, ignore future requests, and remove the potential for long_lived references to memory.

#import <VVCMSAPI.h>
